# RoamWise Autopilot — Run a $1M/yr Operation Solo, Powered by Free AI

An honest note first: no static app "auto-enhances itself" — anyone selling that
is selling vapor. What IS real: a solo founder + Claude + free AI tools running
weekly loops that a 10-person startup used to need. This playbook is that machine.

## THE WEEKLY OPERATING SYSTEM (90 min/week total)

### Monday — PRODUCT (20 min, with Claude)
Open the admin console → Ideas tab → copy the top 3 user requests → paste to Claude:
> "Here are this week's top user requests for RoamWise: [paste]. Pick the
> highest-impact one, implement it in my index.html architecture, and build me
> the new APK + AAB."
Ship the update same day. Demand → deployment in hours is your moat.

### Tuesday — MARKETING CONTENT (20 min)
Prompt Claude: "Write me this week's batch: 3 Instagram Reel scripts
(15-sec, screen-record style) about [feature/destination], 2 tweets for
#buildinpublic with this week's numbers [paste stats], 1 Reddit value-post
for r/IndiaTravel answering a real question with my crowd data."
Batch-record on your phone. Schedule with Buffer (free tier).

### Wednesday — SEO (15 min)
Whenever you add destinations to the app DB, tell Claude: "Regenerate the SEO
guide pages from the updated DB" → upload the new /guides folder to GitHub.
Monthly: "Write 2 new blog posts targeting [keyword from Search Console]."
Check Google Search Console (free) → which queries are rising → feed back.

### Thursday — FINANCE (10 min)
Admin console → Export claims CSV + users CSV → paste to Claude:
> "Here are this week's numbers. Update my running P&L, compute conversion
> rate, ARPU, and flag anything unusual."
Zero accountants until ₹10L/yr.

### Friday — RESEARCH (15 min)
Prompt: "Search the web for: new travel app launches this week, new free AI
APIs I could add as providers, and one growth tactic a competitor used."
One insight per week compounds into an unassailable lead.

### Saturday — COMMUNITY (10 min)
Reply to every review, approve claims, message 3 power users personally.
